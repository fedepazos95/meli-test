# MeLi Test

This is a test app for MeLi  
[Demo here.](https://meli-test-fp95.herokuapp.com/)

#### Getting Started
```
> git clone https://github.com/fedepazos95/meli-test.git
> cd meli-test && yarn
> cd client && yarn
> cd .. && yarn dev
```
